Word count is generated using the TeXcount script, the steps below describe the process. 

1.) Download the TeXCount script at https://app.uio.no/ifi/texcount/download.html
2.) Extract the TeXCount zip file
3.) Run the script (texcount.pl) on the relevant .tex file (dissertation and abstract) with the -sum flag, e.g. $PATH_TO_SCRIPT/texcount.pl -sum acs-dissertation.tex.
4.) Record the Sum count field in the output for each file and sum all them up.
